TECHNICAL DETAILS
════════════════════════════════════════════════════════════════════════════════

ENDPOINT: POST /api/capture-print
- Receives form data as JSON
- Validates required fields
- Calls N8N webhook from server
- Returns success/failure to client

FLOW:
  User clicks Print
    ↓
  Form data POSTs to /api/capture-print
    ↓
  Server validates data
    ↓
  Server calls N8N webhook
    ↓
  N8N saves to Airtable
    ↓
  API returns success (200 OK)
    ↓
  Print dialog opens
    ↓
  ✓ Complete

TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

Q: Print dialog doesn't open?
A: 1. Open DevTools (F12)
   2. Fill form completely (all * fields required)
   3. Click Print
   4. Check Console for errors
   5. Try different browser

Q: Data not in Airtable?
A: 1. Check N8N webhook is active
   2. Test webhook: curl https://n8n.srv1422365.hstgr.cloud/webhook/hn-luxury-capture
   3. Check Vercel Function Logs for errors
   4. Wait 5+ seconds and refresh Airtable

Q: PDF doesn't download?
A: 1. Try different browser
   2. Check browser download settings
   3. Try submitting again
